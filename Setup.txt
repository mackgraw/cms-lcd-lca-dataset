# Setup

This project pulls **Local Coverage Determinations (LCDs)** and **Local Coverage Articles (LCAs)** from the CMS **Coverage API**, shards the workload, merges outputs, computes normalized + day‑over‑day changes, and publishes assets in a daily Release.

## Quick start (local)
```bash
python -m venv .venv && source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python scripts/run_once.py
